**google_seo.py** (294 lines, 5 occurrences at lines ~52, 109, 162, 211, 260):
Each of the 5 functions (get_seo_performance, get_top_search_queries, get_ga4_traffic, get_ga4_page_performance, get_seo_content_gaps) follows the identical pattern:
1. Change `def fn_name(...)` to `async def fn_name(...)`
2. Replace the entire try/except block containing `asyncio.get_event_loop()` + `ThreadPoolExecutor` + `asyncio.run` with a simple `return await underlying_async_fn(...)` wrapped in try/except
3. Remove `import asyncio` from the top-level import if no other usage remains. Remove any `import concurrent.futures` (these are lazy-imported inline, so just remove the inline import blocks)
4. Keep the existing try/except Exception catch with the same error return format

Example transformation for get_seo_performance:
```python
# BEFORE:
def get_seo_performance(...) -> dict[str, Any]:
    ...
    try:
        loop = asyncio.get_event_loop()
        if loop.is_running():
            import concurrent.futures
            with concurrent.futures.ThreadPoolExecutor() as executor:
                future = executor.submit(asyncio.run, search_console_performance(...))
                return future.result(timeout=60)
        else:
            return loop.run_until_complete(search_console_performance(...))
    except Exception as e:
        return {"success": False, "error": str(e)}

# AFTER:
async def get_seo_performance(...) -> dict[str, Any]:
    ...
    try:
        return await search_console_performance(...)
    except Exception as e:
        return {"success": False, "error": str(e)}
```
