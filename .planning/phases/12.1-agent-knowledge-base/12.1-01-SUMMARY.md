## What Was Built

### Task 1: SQL Migration
- `admin_knowledge_entries` table with id, filename, file_type, mime_type, file_path, agent_scope (null=global), uploaded_by, status, chunk_count, embedding_ids, file_size_bytes, error_message, timestamps
- Indexes on agent_scope, status, file_type columns
- RLS enabled (service-role bypass only — no anon access)
- `match_system_knowledge` RPC: cosine similarity search on existing `embeddings` table filtering `metadata->>'scope'='system'`, with optional `filter_agent_scope` to narrow to global + agent-specific entries
- `admin-knowledge` storage bucket (public=false, 100MB file size limit)
- 8 `admin_agent_permissions` rows for knowledge domain: upload_knowledge (confirm/high), delete_knowledge_entry (confirm/medium), and 6 auto/low read operations

### Task 2: knowledge_service.py
- `process_document(file_bytes, filename, mime_type, agent_scope, uploaded_by)`: dispatches to pypdf/python-docx/raw-decode by MIME type, calls `ingest_document` with scope=system metadata, inserts tracking entry
- `process_image(file_bytes, filename, mime_type, agent_scope, uploaded_by)`: Gemini 2.5 Flash vision description, `generate_embeddings_batch`, manual embedding row insert, tracking entry
- `process_video(file_bytes, filename, mime_type, agent_scope, uploaded_by)`: uploads to Storage, inserts entry with status=processing, enqueues `admin_knowledge_video` ai_jobs row — no inline transcription
- `process_video_transcript(entry_id, file_path, agent_scope, mime_type)`: downloads from Storage, `_extract_audio_from_video` via ffmpeg-python in asyncio.to_thread, `transcribe_audio`, `ingest_document`, updates entry to completed; on error updates to failed and re-raises
- `search_system_knowledge(query, agent_name, top_k)`: `generate_embeddings_batch` + `match_system_knowledge` RPC
- `get_knowledge_stats()`: queries all entries, aggregates by agent_scope, returns total_entries, total_embeddings, by_agent dict, storage_bytes

### Task 3: WorkflowWorker Handler
- `from app.services.knowledge_service import process_video_transcript` added at module top-level
- `"admin_knowledge_video": self.handle_admin_knowledge_video` added to handlers dict
- `handle_admin_knowledge_video(input_data)` delegates to `process_video_transcript` with correct kwargs
