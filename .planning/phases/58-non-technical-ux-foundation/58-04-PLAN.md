<objective>
Enable non-technical users to discover and launch workflows via natural language ("I want to launch a product") and browse a visual template gallery of pre-built content types (Product Launch, Newsletter, Testimonial, etc.).

Purpose: Users currently must know exact workflow names or navigate menus. This plan adds NL discovery (type what you want, get matching workflows) and a browsable gallery (see all content types at a glance), both with one-click launch. Depends on Plan 01's suggestions router.

Output: `WorkflowDiscoveryService` + two new API endpoints + `WorkflowLauncher` and `TemplateGallery` frontend components.
</objective>
