       ```python
       async def generate_legal_document(
           doc_type: str,
           business_name: str,
           business_description: str,
           jurisdiction: str = "United States",
           additional_context: str = "",
       ) -> dict:
       ```
       - Validates doc_type is one of: "privacy_policy", "terms_of_service", "refund_policy".
       - Uses `google.genai` (async) to generate the document content. Build a detailed prompt that includes: doc_type, business_name, business_description, jurisdiction, additional_context. The prompt should instruct the model to produce a complete, professional legal document appropriate for the jurisdiction, with standard sections and clear language. Include a disclaimer that this is AI-generated and should be reviewed by legal counsel.
       - Import model config from `app.agents.shared.get_model`.
       - Use `google.genai.Client` with `aio.models.generate_content` for async generation.
       - Return: {"success": True, "document_type": doc_type, "business_name": business_name, "jurisdiction": jurisdiction, "content": generated_text, "disclaimer": "This document was AI-generated and should be reviewed by qualified legal counsel before use."}.
       - On error: {"success": False, "error": str(e)}.
       - Do NOT attempt PDF generation here -- the agent can follow up with generate_pdf_report if the user wants a PDF. Keep it simple: generate the text content.

       **explain_contract_clause:**
       ```python
       async def explain_contract_clause(
           clause_text: str,
           contract_type: str = "general",
       ) -> dict:
       ```
       - Validates clause_text is non-empty after strip.
       - Uses `google.genai` (async) to analyze the clause. Build a prompt that asks for: plain-English explanation (what this clause means in everyday language), key implications (what obligations/rights this creates), risk level (low/medium/high with justification), things to watch for (red flags or negotiation points), and any unusual or non-standard terms.
       - Return: {"success": True, "clause_text": clause_text (truncated to 500 chars for readability), "explanation": str, "implications": list[str], "risk_level": str, "watch_items": list[str]}.
       - On error: {"success": False, "error": str(e)}.
