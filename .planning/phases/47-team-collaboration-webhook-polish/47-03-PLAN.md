    - TestWebhookTools: create_webhook_endpoint inserts row with encrypted secret, validates events against EVENT_CATALOG, returns endpoint id and plaintext secret
    - TestWebhookTools: delete_webhook_endpoint removes endpoint owned by user, returns confirmation message
    - TestWebhookTools: list_webhook_events returns EVENT_CATALOG summary (event_type + description for each)
    - TestWebhookTools: get_webhook_delivery_log returns recent deliveries for a given endpoint_id owned by user
  </behavior>
  <action>
    1. Create `app/agents/tools/webhook_tools.py` following the exact pattern of `communication_tools.py`:
       - Module docstring with exported list documentation
       - Raw async function exports (not FunctionTool) — `sanitize_tools` handles wrapping

       - `async def list_webhook_endpoints(tool_context) -> dict`:
         Extract user_id from `tool_context.state["user_id"]`. Query `webhook_endpoints` table for user's endpoints. Return list with id, url, events, active, description, consecutive_failures. Do NOT return secrets.

       - `async def create_webhook_endpoint(tool_context, url: str, events: list[str], description: str = "") -> dict`:
         Validate events against EVENT_CATALOG keys. Generate secret via `secrets.token_urlsafe(32)`, encrypt with `encrypt_secret()`. Insert row via service client. Return `{endpoint_id, secret, message}`.

       - `async def delete_webhook_endpoint(tool_context, endpoint_id: str) -> dict`:
         Verify ownership (user_id match). Delete row. Return confirmation.

       - `async def list_webhook_events(tool_context) -> dict`:
         Return EVENT_CATALOG as list of `{event_type, description}` dicts. No schema detail needed for chat display.

       - `async def get_webhook_delivery_log(tool_context, endpoint_id: str, limit: int = 20) -> dict`:
         Verify endpoint ownership. Query `webhook_deliveries` for endpoint_id, order by created_at DESC, limit. Return list with event_type, status, attempts, response_code, created_at.

       - Export: `WEBHOOK_TOOLS = [list_webhook_endpoints, create_webhook_endpoint, delete_webhook_endpoint, list_webhook_events, get_webhook_delivery_log]`

    2. Wire into `app/agents/operations/agent.py`:
       - Add import: `from app.agents.tools.webhook_tools import WEBHOOK_TOOLS`
       - Add `*WEBHOOK_TOOLS` to `OPERATIONS_AGENT_TOOLS` list with comment `# Phase 47: Outbound webhook management`

    3. Add TestWebhookTools class to `tests/unit/test_outbound_webhooks.py`. Mock `execute_async`, `get_service_client`, and `tool_context.state["user_id"]`. Verify each tool function returns expected shape. Verify event validation rejects unknown events. Verify ownership check on delete and delivery log.
