<objective>
Create persona-specific behavioral instruction fragments that transform how the ExecutiveAgent and all 10 sub-agents communicate based on the user's persona tier.

Purpose: The existing persona system provides policy metadata (response_style, output_contract) and strategic focus per agent, but lacks concrete behavioral directives that control the agent's actual communication style. A solopreneur should get "Hey, here's what to do next" while an enterprise user should get "Executive Summary: Based on governance requirements..." — this plan creates those behavioral instructions.

Output: A new `behavioral_instructions.py` module with persona-to-behavioral-instruction mappings for Executive + all sub-agents, integrated into the existing `build_persona_policy_block()` pipeline.
