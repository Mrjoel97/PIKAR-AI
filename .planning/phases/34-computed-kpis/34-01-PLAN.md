async def compute_kpis(self, *, user_id: str, persona: str) -> dict[str, Any]:
    # Returns { "persona": str, "kpis": list[dict with label/value/unit] }

Internal methods for each persona — all async, all using `_safe_rows` for safety:

**Solopreneur (`_solopreneur_kpis`)**:
1. **Cash Collected**: Query `invoices` WHERE user_id AND status='paid', JOIN to `orders` to sum `total_amount`. Format as currency. Unit: "currency".
2. **Weekly Pipeline**: Query `contacts` WHERE user_id AND lifecycle_stage IN ('opportunity', 'qualified'), SUM estimated_value. Format as currency. Unit: "currency".
3. **Content Consistency**: Count `content_bundles` WHERE user_id AND created_at >= 7 days ago. Compare to count of prior 7 days. Value: "{this_week} this week" or ratio. Unit: "pieces".

**Startup (`_startup_kpis`)**:
1. **MRR Growth**: Query `orders` WHERE user_id AND status='paid'. Sum total_amount for current month vs prior month. Value: percentage change (e.g., "+12%"). Unit: "percent".
2. **Activation & Conversion**: Count `contacts` WHERE user_id AND lifecycle_stage='customer' divided by total contacts. Value: percentage. Unit: "percent".
3. **Experiment Velocity**: Count `workflow_executions` WHERE user_id AND status='completed' AND completed_at >= 7 days ago. Value: count. Unit: "per week".

**SME (`_sme_kpis`)**:
1. **Department Performance**: Count `departments` WHERE status='RUNNING' divided by total departments. Value: percentage. Unit: "percent".
2. **Process Cycle Time**: Query `workflow_executions` WHERE user_id AND status='completed' AND completed_at IS NOT NULL. Compute avg (completed_at - created_at) in hours. Value: formatted hours. Unit: "hours".
3. **Margin & Compliance**: Count `compliance_risks` WHERE user_id AND status IN ('mitigated', 'resolved') divided by total risks. Value: percentage. Unit: "percent".

**Enterprise (`_enterprise_kpis`)**:
1. **Portfolio Health**: Count `initiatives` WHERE user_id AND status='in_progress' AND progress >= 50 divided by total active initiatives. Value: score 0-100. Unit: "score".
2. **Risk & Control Coverage**: Count `compliance_risks` WHERE user_id AND mitigation_plan IS NOT NULL divided by total risks. Value: percentage. Unit: "percent".
3. **Reporting Quality**: Count `user_reports` WHERE user_id AND created_at >= current month start. Value: count. Unit: "reports".

All percentage values formatted as "XX%" (integer). Currency formatted using `_format_currency` helper (same as DashboardSummaryService). Empty data returns "0" or "$0" or "0%" — never raises.
