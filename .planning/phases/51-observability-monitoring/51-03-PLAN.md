**1. Create migration `supabase/migrations/20260409000000_agent_latency_rollups.sql`:**

Follow the pattern from `20260407000000_stripe_webhook_events.sql` (RLS + service_role policy + DO-block idempotent indexes):

```sql
-- Agent latency rollups for observability dashboard (Phase 51 OBS-02)
-- Hourly pre-computed percentiles consumed by /admin/observability/* endpoints.

CREATE TABLE IF NOT EXISTS agent_latency_rollups (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    agent_name TEXT NOT NULL,
    status TEXT NOT NULL CHECK (status IN ('success', 'error', 'timeout')),
    bucket_start TIMESTAMPTZ NOT NULL,
    bucket_end TIMESTAMPTZ NOT NULL,
    sample_count INTEGER NOT NULL DEFAULT 0,
    p50_ms DOUBLE PRECISION,
    p95_ms DOUBLE PRECISION,
    p99_ms DOUBLE PRECISION,
    error_count INTEGER NOT NULL DEFAULT 0,
    total_duration_ms BIGINT NOT NULL DEFAULT 0,
    created_at TIMESTAMPTZ DEFAULT now()
);

-- Unique constraint for upsert idempotency
DO $$ BEGIN
    IF NOT EXISTS (SELECT 1 FROM pg_indexes WHERE indexname = 'uix_agent_latency_rollups_agent_status_bucket') THEN
        CREATE UNIQUE INDEX uix_agent_latency_rollups_agent_status_bucket
            ON agent_latency_rollups (agent_name, status, bucket_start);
    END IF;
END $$;

-- Query index for dashboard time-range queries
DO $$ BEGIN
    IF NOT EXISTS (SELECT 1 FROM pg_indexes WHERE indexname = 'idx_agent_latency_rollups_bucket_start') THEN
        CREATE INDEX idx_agent_latency_rollups_bucket_start
            ON agent_latency_rollups (bucket_start DESC);
    END IF;
END $$;

-- RLS: service role only
ALTER TABLE agent_latency_rollups ENABLE ROW LEVEL SECURITY;

DO $$ BEGIN
    IF NOT EXISTS (
        SELECT 1 FROM pg_policies
        WHERE tablename = 'agent_latency_rollups' AND policyname = 'service_role_agent_latency_rollups'
    ) THEN
        CREATE POLICY service_role_agent_latency_rollups ON agent_latency_rollups
            FOR ALL USING (auth.role() = 'service_role');
    END IF;
END $$;
```

Required methods (all async, all query `agent_telemetry` or `agent_latency_rollups` via `execute_async`):

a) `compute_latency_percentiles(self, agent_name: str | None, start: datetime, end: datetime) -> dict`:
   - If the window is entirely within the last 24h, query `agent_telemetry` using PostgreSQL `percentile_cont()` via Supabase RPC. Since Supabase Python SDK does not support raw SQL, use a simpler approach: fetch rows with `duration_ms` from `agent_telemetry` filtered by time range and optional agent_name, then compute percentiles in Python using `sorted()` and index math. This is fine for solopreneur-scale data volumes.
   - If the window is entirely older than 24h, query `agent_latency_rollups`.
   - If the window spans the boundary, fetch from both and combine.
   - Return: `{"p50": float, "p95": float, "p99": float, "sample_count": int, "error_count": int}`

b) `compute_error_rate(self, agent_name: str | None, start: datetime, end: datetime) -> dict`:
   - Query `agent_telemetry` for total count and error count in the window.
   - Return: `{"error_rate": float, "error_count": int, "total_count": int}`

c) `compute_ai_cost_by_agent(self, start: datetime, end: datetime) -> dict[str, float]`:
   - Query `agent_telemetry` for `agent_name, input_tokens, output_tokens` in the window.
   - For each row, look up the agent's model in `AI_MODEL_PRICING`. Since `agent_telemetry` does not store model name, use a mapping: the 10 specialized agents use `gemini-2.5-pro` (primary) with fallback to `gemini-2.5-flash`. For cost estimation, use `gemini-2.5-pro` pricing as the default since that is the primary model. This is an approximation — documented in the module docstring.
   - Sum costs per agent_name. Return dict.

d) `compute_ai_cost_by_user(self, start: datetime, end: datetime, top_n: int = 10) -> list[dict]`:
   - Same approach but grouped by user_id. Return top N users by cost.

e) `compute_ai_cost_by_day(self, start: datetime, end: datetime) -> list[dict]`:
   - Same approach but grouped by date(created_at). Return `[{"date": "YYYY-MM-DD", "cost_usd": float}]`.

f) `project_monthly_ai_spend(self) -> dict`:
   - Get last 7 days of daily cost via `compute_ai_cost_by_day()`.
   - Average daily cost, multiply by days remaining in month, add MTD actual.
   - Return: `{"mtd_actual": float, "projected_full_month": float, "projection_method": "linear_7day"}`

g) `run_hourly_rollup(self) -> dict`:
   - Compute the previous hour's bucket from `agent_telemetry`.
   - Group by `(agent_name, status)`, compute percentiles per group.
   - Upsert into `agent_latency_rollups` (conflict on the unique index).
   - Return: `{"buckets_written": int, "bucket_start": str, "bucket_end": str}`

h) `check_error_threshold(self) -> dict | None`:
   - Compute error rate over the last `OBSERVABILITY_THRESHOLD_WINDOW_MINUTES`.
   - If `error_rate > OBSERVABILITY_ERROR_RATE_THRESHOLD`, write to `admin_audit_log` via `log_admin_action()` (import from `app.services.admin_audit`) with `action="observability.threshold_breach"` and details including threshold, observed rate, affected agents, time window.
   - Return the breach details dict if threshold exceeded, else None.

Helper for Python percentile computation:
```python
def _percentile(sorted_values: list[float], p: float) -> float:
    """Compute the p-th percentile from a pre-sorted list."""
    if not sorted_values:
        return 0.0
    k = (len(sorted_values) - 1) * p
    f = int(k)
    c = f + 1
    if c >= len(sorted_values):
        return sorted_values[f]
    return sorted_values[f] + (k - f) * (sorted_values[c] - sorted_values[f])
```

Add a helper function to parse window strings:
```python
def _parse_window(window: str) -> tuple[datetime, datetime]:
    """Parse window string (1h, 24h, 7d, 30d) to (start, end) datetimes."""
    now = datetime.now(tz=timezone.utc)
    if window == "1h": return (now - timedelta(hours=1), now)
    if window == "24h": return (now - timedelta(hours=24), now)
    if window == "7d": return (now - timedelta(days=7), now)
    if window == "30d": return (now - timedelta(days=30), now)
    return (now - timedelta(hours=24), now)  # default 24h
```
