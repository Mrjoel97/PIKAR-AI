    Create `app/agents/admin/tools/integrations.py` following the analytics.py tool pattern exactly.

    **Structure:**

    1. Module docstring explaining the 6 integration tools, all read-only auto tier.

    2. `_check_autonomy(action_name)` — duplicated self-contained helper (per project pattern from users.py, analytics.py — NOT imported from another module). Queries admin_agent_permissions table. Returns None for auto, error dict for blocked, confirmation dict for confirm.

    3. `_get_integration_config(provider)` — shared helper that:
       - Calls `get_service_client().table("admin_integrations").select("api_key_encrypted, config, is_active, base_url").eq("provider", provider).limit(1).execute()`
       - Returns `{"error": "..."}` if no row, not active, or api_key_encrypted is NULL
       - Otherwise decrypts api_key via `decrypt_secret()` and returns `(api_key, config, base_url)`

    4. Six tool functions (all async, all auto tier):

       **`sentry_get_issues(limit: int = 25) -> dict[str, Any]`**
       - Docstring: "Fetch recent Sentry error issues for the configured project. Autonomy tier: auto (read-only)."
       - _check_autonomy("sentry_get_issues")
       - _get_integration_config("sentry")
       - check_session_budget (use a placeholder session_id="admin" for now — Phase 13 adds real sessions)
       - IntegrationProxyService.call("sentry", "get_issues", api_key, config, {"limit": limit})

       **`sentry_get_issue_detail(issue_id: str) -> dict[str, Any]`**
       - Docstring: "Fetch detailed info for a specific Sentry issue including metadata and tags. Autonomy tier: auto (read-only)."
       - Same pattern, operation="get_issue_detail", params={"issue_id": issue_id}

       **`posthog_query_events(limit: int = 50) -> dict[str, Any]`**
       - Docstring: "Fetch recent PostHog events for the configured project. Autonomy tier: auto (read-only)."
       - operation="get_events", params={"limit": limit}

       **`posthog_get_insights() -> dict[str, Any]`**
       - Docstring: "Fetch saved PostHog insights (dashboards/queries). Autonomy tier: auto (read-only)."
       - operation="get_insights"

       **`github_list_prs(state: str = "open") -> dict[str, Any]`**
       - Docstring: "List recent GitHub pull requests. Autonomy tier: auto (read-only)."
       - operation="get_prs", params={"state": state}

       **`github_get_pr_status(pr_number: int) -> dict[str, Any]`**
       - Docstring: "Get status, checks, and review state for a specific GitHub PR. Autonomy tier: auto (read-only)."
       - operation="get_pr_status", params={"pr_number": pr_number}
