<objective>
Create the proactive alert foundation and enriched daily briefing system.

Purpose: PROACT-01 requires users to receive morning briefings without asking. The existing `/scheduled/slack-daily-briefing` only sends pending approvals + upcoming tasks + key metrics. This plan creates:
1. A centralized ProactiveAlertService that dispatches to in-app + Slack/Teams + email
2. A DailyBriefingAggregator that pulls all 4 required sections (approvals, KPI changes, stalled initiatives, deadlines)
3. A proactive_alert_log table for deduplication and audit

Output: Working proactive alert service + enriched daily briefing endpoint
</objective>
