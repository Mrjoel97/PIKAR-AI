type Persona = 'solopreneur' | 'startup' | 'sme' | 'enterprise' | null;

interface PersonaContextType {

  isLoading: boolean;

  userEmail: string | null;
  agentName: string | null;

export function usePersona() {
  const context = useContext(PersonaContext);

}

    **Define types:**
    ```typescript
    type Persona = 'solopreneur' | 'startup' | 'sme' | 'enterprise' | null;

    interface ImpersonationState {
      isActive: boolean;
      targetUserId: string;
      targetUserEmail: string;
      targetPersona: Persona;
      targetAgentName: string | null;
      sessionStartTime: Date;
      timeRemainingMs: number;
      exitImpersonation: () => void;
    }

    interface TargetUser {
      id: string;
      email: string;
      persona: Persona;
      agentName: string | null;
    }
    ```

    **ImpersonationContext** — `createContext<ImpersonationState | null>(null)`

    **ImpersonationProvider component:**
    - Props: `{ children: ReactNode; targetUser: TargetUser }`
    - **Session timer:** On mount, check `sessionStorage` for key `pikar:impersonate:${targetUser.id}:start`. If exists and less than 30 minutes old, use that timestamp. If expired or missing, set a new one. Store the session start time.
    - **Countdown effect:** `useEffect` with `setInterval(1000ms)` that computes `timeRemainingMs = 30 * 60 * 1000 - (Date.now() - sessionStartTime.getTime())`. When `timeRemainingMs <= 0`, redirect to `/admin/users` via `router.push`. Clear interval on unmount.
    - **Exit handler:** `exitImpersonation()` callback — removes sessionStorage key, redirects to `/admin/users`.
    - **PersonaContext override:** Render a `PersonaContext.Provider` (imported from PersonaContext.tsx — the raw context, not the provider component) with static values: `persona: targetUser.persona, setPersona: () => {} (no-op), isLoading: false, userId: targetUser.id, userEmail: targetUser.email, agentName: targetUser.agentName`. This ensures any component calling `usePersona()` inside the impersonation tree gets the target user's data.
    - Wrap children in: `<ImpersonationContext.Provider><PersonaContext.Provider>{children}</PersonaContext.Provider></ImpersonationContext.Provider>`

    **Export `useImpersonation()` hook** — returns `useContext(ImpersonationContext)`.

    **Create `frontend/src/components/admin/ImpersonationBanner.tsx`:**
    - `'use client'` directive
    - Import `useImpersonation` from `@/contexts/ImpersonationContext`

    **Component:**
    - Reads impersonation state from context
    - `position: sticky; top: 0; z-index: 9999` — MUST be above everything, non-dismissible
    - Background: amber-600 (warning color) per CONTEXT.md specific idea
    - Content: "Viewing as: {targetUserEmail} ({targetPersona} persona) -- READ ONLY"
    - Right side: Time remaining display (MM:SS format) + "Exit Impersonation" button
    - Exit button calls `exitImpersonation()` from context which clears sessionStorage and redirects to `/admin/users`
    - NO close/dismiss button — the banner cannot be removed
    - When time remaining is < 5 minutes, banner background changes to red-600 as visual warning
