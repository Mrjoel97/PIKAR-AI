       - Class `InvoiceFollowupService` extending BaseService
       - Method `async get_overdue_invoices(user_id: str) -> list[dict]`:
         a. Query invoices WHERE user_id = user_id AND status IN ('sent', 'overdue') AND due_date < CURRENT_DATE
         b. For each overdue invoice, also update status to 'overdue' if currently 'sent'

            - subject: "Friendly Reminder: Invoice {invoice_number} - Payment Due"
            - recipient: extract customer_email from invoice metadata if available, else "customer"
            - body: Professional template:
              "Hi {customer_name},\n\nI hope this message finds you well. I wanted to follow up regarding Invoice {invoice_number}, which was due on {due_date} ({days_overdue} days ago).\n\nThe outstanding amount is {amount} {currency}. If payment has already been sent, please disregard this message.\n\nPlease let me know if you have any questions or if there's anything I can help with.\n\nBest regards"

       - Method `async get_overdue_invoices_with_drafts(user_id: str) -> list[dict]`:
         a. Gets overdue invoices, generates draft for each
         b. Returns list of dicts combining invoice info + email draft

       - Method `async get_ytd_revenue(user_id: str) -> float`:

         c. Compute quarterly_payment = estimated_tax / 4
         d. Find next_deadline from QUARTER_DEADLINES
         e. Return: {"ytd_revenue": float, "estimated_annual_tax": float, "quarterly_payment": float, "tax_rate": float, "next_deadline": str, "currency": "USD", "explanation": "Based on ${ytd_revenue} YTD revenue at {rate}% estimated rate, your quarterly estimated tax payment is ${quarterly_payment}. Next deadline: {next_deadline}."}

         a. Check if current date is within REMINDER_WINDOW_DAYS of any quarter deadline

          ```python
          # Overdue invoice follow-ups (FIN-03)
          try:
              followup_svc = InvoiceFollowupService()
              overdue_items = await followup_svc.get_overdue_invoices_with_drafts(user_id)
              if overdue_items:
                  briefing["overdue_invoices"] = {
                      "count": len(overdue_items),
                      "items": overdue_items,
                      "summary": f"{len(overdue_items)} overdue invoice(s) need follow-up"
                  }
          except Exception as exc:
              logger.warning("Failed to fetch overdue invoices for briefing: %s", exc)
          ```

       b. Tax reminder section (only when near quarter deadline):
          ```python
          # Quarterly tax reminders (FIN-05)
          try:
              tax_svc = TaxReminderService()
              if tax_svc.is_reminder_due():
                  estimate = await tax_svc.get_quarterly_tax_estimate(user_id)
                  briefing["tax_reminder"] = {
                      "estimate": estimate,
                      "summary": f"Quarterly estimated tax payment of ${estimate['quarterly_payment']:,.2f} due {estimate['next_deadline']}"
                  }
          except Exception as exc:
              logger.warning("Failed to compute tax reminder for briefing: %s", exc)
          ```
