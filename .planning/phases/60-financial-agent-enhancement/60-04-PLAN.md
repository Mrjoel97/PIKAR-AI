         c. If 3-6 months: use simple linear regression on revenue and expenses, confidence="medium"
         d. If 6+ months: use weighted linear regression (recent months weighted more), confidence="high"
         e. For each projected month:
            - projected_revenue = regression_intercept + regression_slope * month_index
            - projected_expenses = expense_regression_intercept + expense_regression_slope * month_index
            - Clamp revenue and expenses to >= 0
            - net = revenue - expenses
         f. Return: {"title": title, "forecast_months": [...], "confidence": str, "data_months_used": int, "methodology": str, "currency": "USD", "generated_at": ISO timestamp}
