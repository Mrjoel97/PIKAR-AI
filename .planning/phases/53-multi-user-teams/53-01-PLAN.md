   a. Update `create_invite_link()` to accept optional `invited_email: str | None = None` parameter:

   b. Add `get_pending_invites(workspace_id: str) -> list[dict]`:
      - Query `workspace_invites` where `workspace_id` matches, `is_active=True`, `accepted_by IS NULL`
      - Select: id, token, role, invited_email, created_by, expires_at, created_at, is_active
      - Order by created_at DESC
      - Join with user_profiles on created_by to get inviter name (best-effort, same pattern as get_workspace_members)
   c. Add `revoke_invite(invite_id: str, workspace_id: str, actor_user_id: str) -> bool`:
      - Verify actor is admin via get_member_role
      - Update workspace_invites SET is_active=False WHERE id=invite_id AND workspace_id=workspace_id
      - Return True on success
   d. Add `resend_invite(invite_id: str, workspace_id: str, actor_user_id: str) -> dict`:
      - Verify actor is admin
      - Generate new token via secrets.token_urlsafe(32)
      - Update: SET token=new_token, expires_at=now()+7days, is_active=True
      - Return the updated invite record (including invited_email for the caller to re-send email)
