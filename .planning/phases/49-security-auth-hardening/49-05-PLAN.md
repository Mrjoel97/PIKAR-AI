<critical_decisions>
1. **New endpoint at `/admin/governance-audit-log`** — distinct from the existing
   `/admin/audit-log` (which serves admin_audit_log). Two tables, two endpoints, two UI
   tabs. Do NOT merge or alias.

2. **Email resolution strategy** — reuse the same `_resolve_user_emails` helper pattern
   from `app/routers/admin/audit.py`. Resolution is best-effort: if `auth.admin.get_user_by_id`
   fails for a row, fall back to the raw UUID rather than crashing the page.

3. **Filter parameters:**
   - `user_id` (UUID) — exact match on `user_id` column
   - `email` (string, case-insensitive) — resolves to `user_id` via auth.admin.list_users
     OR (faster) joins through user_profiles. PREFERRED PATH: a single auth.admin.list_users
     call to build a {email_lower → user_id} map, then filter.
   - `action_type` (string) — exact match on `action_type` column
   - `start_date` (ISO 8601) — `created_at >= start_date`
   - `end_date` (ISO 8601) — `created_at <= end_date`
   - `limit` (int 1-200, default 50)
   - `offset` (int >= 0, default 0)

4. **No write endpoint** — this is read-only. Plan 04's middleware is the only writer.

5. **Frontend page lives at `/admin/audit-log/governance`** — a sibling under the existing
   audit-log directory, NOT a replacement. The current `(admin)/audit-log/page.tsx` stays
   as the admin-action viewer.

6. **Reuse the existing admin layout guard** — `frontend/src/app/(admin)/layout.tsx`
   already does the server-side `require_admin` check via `/admin/check-access`. The new
   page inherits that guard automatically.

7. **Action type dropdown** — populate by enumerating Plan 04's AUDITED_ROUTES at
   build time. To avoid coupling, the backend exposes a tiny helper endpoint
   `GET /admin/governance-audit-log/actions` returning a sorted list of distinct
   action_type values currently in the table (Postgres `SELECT DISTINCT action_type`).
   The frontend fetches this once on mount and uses it to populate the filter dropdown.

8. **Rate limit: 120/minute** — match the existing admin audit endpoint.
</critical_decisions>

    # Copyright (c) 2024-2026 Pikar AI. All rights reserved.
    # Proprietary and confidential. See LICENSE file for details.

    """Admin governance audit log viewer endpoint.

    Provides:
    - GET /admin/governance-audit-log — paginated, filterable user-action audit log
    - GET /admin/governance-audit-log/actions — distinct action_type list for filter UI

    Queries the governance_audit_log table (populated by AuditLogMiddleware in Phase 49
    Plan 04) via a service-role client. Requires admin authentication via require_admin.

    Each returned entry includes an actor_email field resolved from user_id via the
    Supabase auth admin API. Resolution is best-effort with fallback to raw UUID.
    """

    from __future__ import annotations

    import asyncio
    import logging

    from fastapi import APIRouter, Depends, HTTPException, Request

    from app.middleware.admin_auth import require_admin
    from app.middleware.rate_limiter import limiter
    from app.services.supabase import get_service_client

    logger = logging.getLogger(__name__)

    router = APIRouter()


    async def _resolve_actor_emails(client, rows: list[dict]) -> list[dict]:
        """Resolve user_id UUIDs to emails for each governance_audit_log row."""
        unique_ids: set[str] = {
            row["user_id"] for row in rows if row.get("user_id")
        }

        async def _resolve_email(uid: str) -> tuple[str, str]:
            try:
                response = await asyncio.to_thread(
                    client.auth.admin.get_user_by_id, uid
                )
                email = (
                    response.user.email
                    if response and response.user
                    else None
                )
                return uid, email or uid
            except Exception:
                logger.warning(
                    "Could not resolve user_id %s to email; using raw UUID", uid
                )
                return uid, uid

        if unique_ids:
            resolved = await asyncio.gather(*[_resolve_email(uid) for uid in unique_ids])
            id_to_email = dict(resolved)
        else:
            id_to_email = {}

        return [
            {**row, "actor_email": id_to_email.get(row.get("user_id"), "Unknown")}
            for row in rows
        ]


    @router.get("/governance-audit-log")
    @limiter.limit("120/minute")
    async def list_governance_audit_log(
        request: Request,
        admin_user: dict = Depends(require_admin),  # noqa: B008
        user_id: str | None = None,
        email: str | None = None,
        action_type: str | None = None,
        start_date: str | None = None,
        end_date: str | None = None,
        limit: int = 50,
        offset: int = 0,
    ) -> dict:
        """Return paginated, filterable governance_audit_log entries."""
        if limit < 1 or limit > 200:
            raise HTTPException(status_code=400, detail="limit must be between 1 and 200")
        if offset < 0:
            raise HTTPException(status_code=400, detail="offset must be 0 or greater")

        client = get_service_client()

        # If filtering by email, resolve to user_id first via list_users
        target_user_id = user_id
        if email and not target_user_id:
            try:
                users_resp = await asyncio.to_thread(
                    client.auth.admin.list_users
                )
                email_lower = email.strip().lower()
                for u in (users_resp or []):
                    if (getattr(u, "email", "") or "").lower() == email_lower:
                        target_user_id = u.id
                        break
                if target_user_id is None:
                    return {"entries": [], "total": 0, "limit": limit, "offset": offset}
            except Exception as exc:
                logger.error("email->user_id resolve failed: %s", exc)
                raise HTTPException(
                    status_code=503, detail="User lookup unavailable"
                ) from exc

        try:
            query = (
                client.table("governance_audit_log")
                .select("*", count="exact")
                .order("created_at", desc=True)
            )

            if target_user_id:
                query = query.eq("user_id", target_user_id)
            if action_type:
                query = query.eq("action_type", action_type)
            if start_date:
                query = query.gte("created_at", start_date)
            if end_date:
                query = query.lte("created_at", end_date)

            query = query.range(offset, offset + limit - 1)
            result = query.execute()
            total = result.count if result.count is not None else len(result.data)

            entries = await _resolve_actor_emails(client, result.data)

            return {
                "entries": entries,
                "total": total,
                "limit": limit,
                "offset": offset,
            }
        except HTTPException:
            raise
        except Exception as exc:
            logger.error("Failed to query governance_audit_log: %s", exc, exc_info=True)
            raise HTTPException(
                status_code=500, detail="Failed to retrieve governance audit log entries"
            ) from exc


    @router.get("/governance-audit-log/actions")
    @limiter.limit("60/minute")
    async def list_distinct_actions(
        request: Request,
        admin_user: dict = Depends(require_admin),  # noqa: B008
    ) -> dict:
        """Return the sorted distinct list of action_type values for filter UI."""
        client = get_service_client()
        try:
            # Use a raw RPC or fall back to fetching a window and deduping in Python.
            # Simpler approach: select action_type with limit 5000, dedupe in Python.
            result = (
                client.table("governance_audit_log")
                .select("action_type")
                .limit(5000)
                .execute()
            )
            actions = sorted({r["action_type"] for r in result.data if r.get("action_type")})
            return {"actions": actions}
        except Exception as exc:
            logger.error("Failed to fetch distinct actions: %s", exc, exc_info=True)
            raise HTTPException(
                status_code=500, detail="Failed to fetch action types"
            ) from exc

    'use client';

    // Copyright (c) 2024-2026 Pikar AI. All rights reserved.
    // Proprietary and confidential. See LICENSE file for details.

    /**
     * GovernanceAuditTable
     *
     * Filterable, paginated table for the user-action audit trail
     * (governance_audit_log). Reads via GET /admin/governance-audit-log.
     */

    import React, { useEffect, useState, useCallback } from 'react';
    import { fetchWithAuth } from '@/services/api';

    interface AuditEntry {
      id: string;
      user_id: string;
      actor_email: string;
      action_type: string;
      resource_type: string;
      resource_id: string | null;
      details: Record<string, unknown>;
      ip_address: string | null;
      created_at: string;
    }

    interface Filters {
      email: string;
      action_type: string;
      start_date: string;
      end_date: string;
    }

    const PAGE_SIZE = 50;

    export function GovernanceAuditTable() {
      const [entries, setEntries] = useState<AuditEntry[]>([]);
      const [total, setTotal] = useState(0);
      const [offset, setOffset] = useState(0);
      const [loading, setLoading] = useState(false);
      const [error, setError] = useState<string | null>(null);
      const [actionsList, setActionsList] = useState<string[]>([]);
      const [filters, setFilters] = useState<Filters>({
        email: '',
        action_type: '',
        start_date: '',
        end_date: '',
      });

      // Fetch the distinct action_type list once on mount.
      useEffect(() => {
        fetchWithAuth('/admin/governance-audit-log/actions')
          .then((r) => r.json())
          .then((d) => setActionsList(d.actions || []))
          .catch(() => setActionsList([]));
      }, []);

      const fetchEntries = useCallback(async () => {
        setLoading(true);
        setError(null);
        try {
          const params = new URLSearchParams();
          params.set('limit', String(PAGE_SIZE));
          params.set('offset', String(offset));
          if (filters.email) params.set('email', filters.email);
          if (filters.action_type) params.set('action_type', filters.action_type);
          if (filters.start_date) params.set('start_date', filters.start_date);
          if (filters.end_date) params.set('end_date', filters.end_date);

          const r = await fetchWithAuth(`/admin/governance-audit-log?${params}`);
          if (!r.ok) throw new Error(`HTTP ${r.status}`);
          const data = await r.json();
          setEntries(data.entries || []);
          setTotal(data.total || 0);
        } catch (e) {
          setError(e instanceof Error ? e.message : 'Failed to load audit log');
        } finally {
          setLoading(false);
        }
      }, [filters, offset]);

      useEffect(() => {
        fetchEntries();
      }, [fetchEntries]);

      const onFilterChange = (key: keyof Filters, value: string) => {
        setFilters((f) => ({ ...f, [key]: value }));
        setOffset(0);
      };

      return (
        <div className="space-y-4">
          {/* Filter bar */}
          <div className="grid grid-cols-1 md:grid-cols-4 gap-3 p-4 bg-gray-800 rounded-xl">
            <input
              type="text"
              placeholder="Filter by user email"
              value={filters.email}
              onChange={(e) => onFilterChange('email', e.target.value)}
              className="px-3 py-2 bg-gray-900 border border-gray-700 rounded-lg text-sm text-gray-100"
              data-testid="filter-email"
            />
            <select
              value={filters.action_type}
              onChange={(e) => onFilterChange('action_type', e.target.value)}
              className="px-3 py-2 bg-gray-900 border border-gray-700 rounded-lg text-sm text-gray-100"
              data-testid="filter-action-type"
            >
              <option value="">All actions</option>
              {actionsList.map((a) => (
                <option key={a} value={a}>{a}</option>
              ))}
            </select>
            <input
              type="date"
              value={filters.start_date}
              onChange={(e) => onFilterChange('start_date', e.target.value)}
              className="px-3 py-2 bg-gray-900 border border-gray-700 rounded-lg text-sm text-gray-100"
              data-testid="filter-start-date"
            />
            <input
              type="date"
              value={filters.end_date}
              onChange={(e) => onFilterChange('end_date', e.target.value)}
              className="px-3 py-2 bg-gray-900 border border-gray-700 rounded-lg text-sm text-gray-100"
              data-testid="filter-end-date"
            />
          </div>

          {/* Status */}
          {loading && <div className="text-sm text-gray-400">Loading…</div>}
          {error && <div className="text-sm text-rose-400">Error: {error}</div>}

          {/* Table */}
          <div className="overflow-x-auto rounded-xl border border-gray-700">
            <table className="min-w-full text-sm">
              <thead className="bg-gray-800">
                <tr className="text-left text-gray-300">
                  <th className="px-4 py-3">Timestamp</th>
                  <th className="px-4 py-3">Actor</th>
                  <th className="px-4 py-3">Action</th>
                  <th className="px-4 py-3">Resource</th>
                  <th className="px-4 py-3">Details</th>
                </tr>
              </thead>
              <tbody className="bg-gray-900 divide-y divide-gray-800">
                {entries.map((entry) => (
                  <tr key={entry.id} data-testid="audit-row">
                    <td className="px-4 py-3 text-gray-400 whitespace-nowrap">
                      {new Date(entry.created_at).toLocaleString()}
                    </td>
                    <td className="px-4 py-3 text-gray-200">{entry.actor_email}</td>
                    <td className="px-4 py-3 text-indigo-300 font-mono">{entry.action_type}</td>
                    <td className="px-4 py-3 text-gray-400">
                      {entry.resource_type}
                      {entry.resource_id && (
                        <span className="text-gray-600">/{entry.resource_id}</span>
                      )}
                    </td>
                    <td className="px-4 py-3 text-gray-500 max-w-xs truncate">
                      <code className="text-xs">{JSON.stringify(entry.details)}</code>
                    </td>
                  </tr>
                ))}
                {!loading && entries.length === 0 && (
                  <tr>
                    <td colSpan={5} className="px-4 py-12 text-center text-gray-500">
                      No audit entries match the current filters.
                    </td>
                  </tr>
                )}
              </tbody>
            </table>
          </div>

          {/* Pagination */}
          <div className="flex items-center justify-between text-sm text-gray-400">
            <div>
              Showing {entries.length === 0 ? 0 : offset + 1}–{offset + entries.length} of {total}
            </div>
            <div className="flex gap-2">
              <button
                type="button"
