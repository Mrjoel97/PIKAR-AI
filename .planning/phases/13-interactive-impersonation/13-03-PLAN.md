DELETE /admin/impersonate/sessions/{sessionId}
- Auth: Bearer token (any admin)
- Response 200: { "success": true }

    7. Pass `mode` and `sessionToken` through the `impersonationState` object.

    Add `const API_URL = process.env.NEXT_PUBLIC_API_URL || 'http://localhost:8000';` at the top of the file.
