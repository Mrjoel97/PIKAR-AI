**Helper function (not exported):**
```python
async def _detect_provider(provider: str | None = None) -> tuple[str, str | None]:
    """Auto-detect PM provider. Returns (provider, error_message)."""
    if provider:
        return provider, None
    # Check which PM tools are connected
    from app.services.integration_manager import IntegrationManager
    from app.services.request_context import get_current_user_id
    mgr = IntegrationManager()
    uid = get_current_user_id()
    linear = await mgr.get_credentials(uid, "linear") 
    asana = await mgr.get_credentials(uid, "asana")
    if linear and not asana: return "linear", None
    if asana and not linear: return "asana", None
    if linear and asana: return None, "Both Linear and Asana are connected. Please specify: provider='linear' or provider='asana'"
    return None, "No PM tool connected. Connect Linear or Asana in Settings."
```

**Tools:**

1. `async def get_pm_projects(provider: str = None) -> dict` — Auto-detect provider. For linear: call LinearService.list_teams(). For asana: list workspaces then projects. Return project list with id, name.

2. `async def list_pm_tasks(project: str = None, status: str = None, provider: str = None) -> dict` — Auto-detect provider. Query synced_tasks with optional filters. If no tasks synced yet, suggest "Connect and sync projects first in Settings." Return list of tasks.

3. `async def create_pm_task(title: str, description: str, project: str, provider: str = None, priority: str = "medium", labels: list[str] = None) -> dict` — Auto-detect provider. Create both:
   - External issue via LinearService.create_issue() or AsanaService.create_task()
   - Local synced_tasks record with external_id linked
   - Set Redis skip-flag before creation (prevents webhook from double-creating)
   - Return {"success": True, "task": {...}, "external_url": "https://linear.app/..."}

4. `async def update_pm_task(task_id: str, status: str = None, title: str = None, description: str = None, priority: str = None) -> dict` — Read synced_tasks record to get provider + external_id. Call PMSyncService.sync_to_external() which handles status mapping and Redis skip-flag. Return updated task.

5. `async def get_pm_sync_status(provider: str = None) -> dict` — Auto-detect provider. Return sync config (selected projects, last sync time, task count per project, connection status).

Export as:
```python
PM_TASK_TOOLS = [get_pm_projects, list_pm_tasks, create_pm_task, update_pm_task, get_pm_sync_status]
```

All tools use lazy imports and `get_current_user_id()`.
