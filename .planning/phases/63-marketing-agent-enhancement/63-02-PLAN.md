  <action>
Create `app/services/cross_channel_attribution_service.py` with class `CrossChannelAttributionService`:

1. Method `async def get_attribution(self, user_id: str, days: int = 30) -> dict`:
   - Aggregates data from multiple sources into a unified channel view:
     - **Google Ads**: Query ad_spend_tracking for all google_ads campaigns via AdSpendTrackingService
     - **Meta Ads**: Same for meta_ads campaigns
     - **Email**: Query campaigns table for email-type campaigns + UTM-tracked conversions from Shopify orders where utm_source matches email campaigns
     - **Organic/Direct**: Shopify revenue minus attributed revenue from paid channels
   - Returns:
     ```python
     {
       "channels": [
         {"channel": "google_ads", "spend": 1200, "conversions": 40, "revenue": 3600, "roas": 3.0, "cpa": 30, "share_of_revenue_pct": 45},
         {"channel": "meta_ads", "spend": 800, "conversions": 50, "revenue": 4000, "roas": 5.0, "cpa": 16, "share_of_revenue_pct": 50},
         ...
       ],
       "totals": {"total_spend": ..., "total_revenue": ..., "blended_roas": ...},
       "period": {"start": "2026-03-10", "end": "2026-04-09", "days": 30},
       "summary_text": "Meta Ads is your best performer at 5.0x ROAS ($16/customer). Google Ads returns 3.0x ($30/customer)."
     }
     ```

2. Method `async def get_budget_recommendation(self, user_id: str, days: int = 30) -> dict`:
   - Calls get_attribution() to get channel data
   - Identifies highest-ROAS and lowest-ROAS channels (among those with spend > 0)
   - Calculates recommended daily budget shift: min(20% of low-ROAS channel daily spend, $50/day) to avoid drastic changes
   - Returns:
     ```python
     {
       "recommendation_text": "Meta Ads gives 2x better return than Google Ads -- shift $30/day from Google to Meta?",
       "shift_from": {"channel": "google_ads", "current_daily": 40, "recommended_daily": 10},
       "shift_to": {"channel": "meta_ads", "current_daily": 27, "recommended_daily": 57},
       "expected_impact": "Could gain ~6 more conversions/week at current Meta ROAS",
       "channels": [...],  # full attribution data
       "action_available": True  # whether shift can be applied
     }
     ```
   - If only one channel or all channels have similar ROAS (within 10%), return "Budget allocation is well-balanced" with action_available=False

3. Helper `_aggregate_ad_channel(self, user_id, platform, days)` -- fetches all campaigns for a platform and sums their spend tracking data.

4. Helper `_get_email_attribution(self, user_id, days)` -- queries campaigns with campaign_type='email' and sums their recorded metrics.

5. Helper `_get_organic_revenue(self, user_id, days, attributed_revenue)` -- attempts Shopify analytics total revenue minus attributed, falls back to 0 if Shopify not connected.

Write tests mocking AdCampaignService, AdSpendTrackingService, and Supabase queries. Verify ROAS calculations, share-of-revenue percentages, and recommendation text.
