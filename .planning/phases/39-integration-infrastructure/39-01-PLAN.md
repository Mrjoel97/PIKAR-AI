  <action>
    1. **Integrations router** (`app/routers/integrations.py`):
       Create `APIRouter(prefix="/integrations", tags=["Integrations"])`.

       Endpoints:

       **`GET /integrations/providers`** (public, no auth needed):
       Returns list of all providers from `PROVIDER_REGISTRY` with fields: `key`, `name`, `auth_type`, `category`, `icon_url`, `scopes`. Do NOT include env var names or secrets.

       **`GET /integrations/{provider}/authorize`** (auth required):
       - Look up provider in `PROVIDER_REGISTRY`, 404 if not found, 400 if not oauth2
       - Get `user_id` from `get_current_user_id(request)` (import from `app/routers/onboarding`)
       - Generate CSRF state token: `f"{user_id}:{provider}:{secrets.token_urlsafe(16)}"`
       - Store state in Redis with 600s TTL using cache service `set_generic` (key: `pikar:integration:oauth_state:{state}`)
       - Build auth URL with params: `client_id` (from env), `redirect_uri` (callback URL), `response_type=code`, `scope` (joined scopes), `state`, `access_type=offline`, `prompt=consent`
       - Return `RedirectResponse` to auth URL

       **`GET /integrations/{provider}/callback`** (no auth — popup returns here):
       - Extract `code` and `state` from query params
       - Validate state from Redis (get + delete atomically), 403 if invalid
       - Parse `user_id` and `provider` from state string
       - Exchange auth code for tokens using httpx POST to provider's `token_url` with `client_id`, `client_secret`, `code`, `redirect_uri`, `grant_type=authorization_code`
       - Store credentials via `IntegrationManager.store_credentials()` (uses service role client since callback has no user JWT)
       - Return HTML page that calls `window.opener.postMessage({type: 'oauth-callback', provider, success: true}, '*')` then `window.close()` — this signals the parent page to refresh

       **`GET /integrations/status`** (auth required):
       - Get `user_id` from request
       - Call `IntegrationManager.get_integration_status(user_id)` 
       - Returns list of `{provider, name, category, connected, account_name, last_sync_at, error_count, last_error}` — one entry per provider in registry, merged with user's credentials

       **`DELETE /integrations/{provider}`** (auth required):
       - Get `user_id` from request
       - Call `IntegrationManager.delete_credentials(user_id, provider)`
       - Return 200 with `{disconnected: true, provider}`

       Use `from app.routers.onboarding import get_current_user_id` for auth.
       Use `from app.services.cache import cache_service` for Redis state storage.
       For the callback endpoint, use `AdminService` pattern (service role) since the OAuth popup callback has no user JWT — the user_id comes from the state token validated from Redis.

    2. **Mount router** in `app/fast_api_app.py`:
       Add `from app.routers.integrations import router as integrations_router` and `app.include_router(integrations_router)` following the existing pattern (alphabetical placement among other router includes).
