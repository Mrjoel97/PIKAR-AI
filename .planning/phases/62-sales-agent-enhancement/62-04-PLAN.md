**Add service methods to `app/services/hubspot_service.py`:**

1. `async def update_contact_score(self, user_id, contact_id, score, qualification_data)`:
   - Read contact from DB, get hubspot_contact_id.
   - Push `hs_lead_status` and custom `pikar_lead_score` property to HubSpot via SDK.
   - Update local contact metadata with score.
   - Set skip flag to prevent webhook echo.

2. `async def add_deal_note(self, user_id, deal_id, note_text, stage_change=None)`:
   - Read deal from `hubspot_deals` table.
   - Create a HubSpot engagement (note) associated with the deal via `hs_client.crm.objects.notes.basic_api.create()`.
   - If `stage_change` provided, also update deal stage via `push_deal_to_hubspot()`.
   - Update `last_activity_at` on local deal row.
   - Return confirmation dict.

**Add tools to `app/agents/tools/hubspot_tools.py`:**

1. `async def score_hubspot_lead(contact_name_or_email: str, score: int, framework: str = "BANT", qualification_notes: str = "") -> dict`:
   - Look up contact in local DB by name/email.
   - If HubSpot connected: call `HubSpotService().update_contact_score()` to push score.
   - If not connected: update local contact metadata only.
   - Log activity in `contact_activities` table (activity_type='lead_scored').
   - Return `{"success": True, "contact": {...}, "score": N, "synced_to_hubspot": bool}`.

2. `async def query_hubspot_crm(query_type: str = "contacts", lifecycle_stage: str | None = None, source: str | None = None, limit: int = 50) -> dict`:
   - Query local `contacts` or `hubspot_deals` table based on `query_type` ("contacts" or "deals").
   - Apply optional filters: lifecycle_stage, source, pipeline, stage.
   - Include aggregations: total count, by-stage breakdown, total value (for deals).
   - Return `{"success": True, "results": [...], "count": N, "aggregations": {...}}`.

3. `async def sync_deal_notes(deal_name_or_id: str, notes: str, next_steps: list[str] | None = None, stage_change: str | None = None) -> dict`:
   - Look up deal in `hubspot_deals` by name (ilike) or ID.
   - Format notes with next_steps as bullet points.
   - If HubSpot connected: call `HubSpotService().add_deal_note()` to push notes and optional stage change.
   - If not connected: update local deal properties JSONB with notes.
   - Update `last_activity_at` timestamp.
   - Return `{"success": True, "deal": {...}, "synced_to_hubspot": bool, "stage_changed": bool}`.
