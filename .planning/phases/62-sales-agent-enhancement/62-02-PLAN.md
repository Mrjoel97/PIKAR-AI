  <action>
Create `app/agents/tools/pipeline_dashboard.py` with two async tool functions:

**Tool 1: get_pipeline_recommendations**
```python
async def get_pipeline_recommendations(
    days_stalled_threshold: int = 14,
    pipeline: str | None = None,
) -> dict[str, Any]:
```
Implementation:
1. Get user_id from request context.
2. Query `hubspot_deals` table for all deals belonging to user, optionally filtered by pipeline.
3. Classify each deal into categories:
   - **stalled**: `updated_at` older than `days_stalled_threshold` days AND stage is not closedwon/closedlost
   - **at_risk**: close_date within 14 days but stage is early (not closedwon), OR amount is significantly lower than pipeline average
   - **healthy**: everything else that is not closed
   - **won**: stage contains "closedwon" or "won"
   - **lost**: stage contains "closedlost" or "lost"
4. For each stalled/at_risk deal, generate specific action recommendation:
   - Stalled: "Send re-engagement email to {contact}", "Offer limited-time discount", "Escalate to manager"
   - At-risk: "Schedule urgent review call", "Prepare competitive comparison", "Offer extended trial"
5. Return `{"success": True, "pipeline_health": {"stalled": [...], "at_risk": [...], "healthy": [...], "won": [...], "lost": [...]}, "recommendations": [...], "summary": {"total_deals": N, "total_value": X, "stalled_count": N, "at_risk_count": N}}`.

**Tool 2: get_lead_attribution**
```python
async def get_lead_attribution(
    period_days: int = 90,
) -> dict[str, Any]:
```
Implementation:
1. Get user_id from request context.
2. Query `contacts` table for contacts created in the last `period_days` days.
3. Group by `source` field, count contacts per source.
4. For each source, calculate conversion rate: contacts that reached 'customer' lifecycle_stage / total.
5. If `utm_source` or `campaign_id` data exists, include campaign-level breakdown.
6. Return `{"success": True, "attribution": {"by_source": [...], "by_campaign": [...], "total_leads": N, "total_converted": N, "overall_conversion_rate": float}, "period_days": N}`.

Use `AdminService` + `execute_async` pattern consistent with hubspot_tools.py. Export as `PIPELINE_DASHBOARD_TOOLS = [get_pipeline_recommendations, get_lead_attribution]`.

For tests: Mock `execute_async` to return sample deal/contact data. Test classification logic, recommendation generation, and attribution grouping.
