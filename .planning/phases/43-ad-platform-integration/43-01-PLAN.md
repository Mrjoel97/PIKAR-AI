**File 2: app/services/google_ads_service.py** -- New file following StripeSyncService pattern.

Create `GoogleAdsService(BaseService)` with these methods:

1. `async def _get_headers(self, user_id: str) -> dict` -- Gets OAuth token via IntegrationManager.get_valid_token(user_id, "google_ads"). Returns headers dict with: `Authorization: Bearer {token}`, `developer-token: {GOOGLE_ADS_DEVELOPER_TOKEN from env}`, `Content-Type: application/json`. If `GOOGLE_ADS_LOGIN_CUSTOMER_ID` env var set, add `login-customer-id` header.

2. `async def get_accessible_customers(self, user_id: str) -> list[str]` -- GET `https://googleads.googleapis.com/v17/customers:listAccessibleCustomers`. Returns list of customer resource names. Used during initial setup to let user pick their ad account.

3. `async def list_campaigns(self, user_id: str, customer_id: str) -> list[dict]` -- POST GAQL search query to `https://googleads.googleapis.com/v17/customers/{customer_id}/googleAds:searchStream`. Query: `SELECT campaign.id, campaign.name, campaign.status, campaign.advertising_channel_type, campaign_budget.amount_micros, campaign.start_date, campaign.end_date FROM campaign WHERE campaign.status != 'REMOVED' ORDER BY campaign.id`. Parse the streaming response, map to dicts with: id, name, status (ENABLED->active, PAUSED->paused), type, budget (micros / 1_000_000), start_date, end_date.

4. `async def create_campaign(self, user_id: str, customer_id: str, name: str, budget_amount: float, campaign_type: str = "SEARCH", status: str = "PAUSED") -> dict` -- 
   - First create a campaign budget: POST `customers/{customer_id}/campaignBudgets:mutate` with `amount_micros: int(budget_amount * 1_000_000)`, `delivery_method: STANDARD`
   - Then create campaign: POST `customers/{customer_id}/campaigns:mutate` with `name`, `campaign_budget` (resource name from above), `advertising_channel_type`, `status: PAUSED` (always start paused — activation requires approval)
   - Return dict with campaign resource name, id, budget

5. `async def update_campaign_status(self, user_id: str, customer_id: str, campaign_id: str, status: str) -> dict` -- POST `customers/{customer_id}/campaigns:mutate` with update operation. Map status: "active" -> "ENABLED", "paused" -> "PAUSED".

6. `async def update_campaign_budget(self, user_id: str, customer_id: str, campaign_budget_id: str, new_amount: float) -> dict` -- POST `customers/{customer_id}/campaignBudgets:mutate` with updated `amount_micros`.

7. `async def get_campaign_performance(self, user_id: str, customer_id: str, start_date: str, end_date: str) -> list[dict]` -- GAQL query: `SELECT campaign.id, campaign.name, metrics.impressions, metrics.clicks, metrics.conversions, metrics.cost_micros, metrics.conversions_value, segments.date FROM campaign WHERE segments.date BETWEEN '{start_date}' AND '{end_date}' AND campaign.status != 'REMOVED'`. Map cost_micros to dollars. Return list of daily performance rows.

All HTTP calls use `httpx.AsyncClient` with 30s timeout. Wrap in try/except, log errors, return structured error dicts on failure. Use lazy import for `httpx`.

**File 3: app/services/meta_ads_service.py** -- New file, same pattern.

Create `MetaAdsService(BaseService)` with these methods:

1. `async def _get_token(self, user_id: str) -> str` -- Gets OAuth token via IntegrationManager.get_valid_token(user_id, "meta_ads"). Raise ValueError if not connected.

2. `async def get_ad_accounts(self, user_id: str) -> list[dict]` -- GET `https://graph.facebook.com/v19.0/me/adaccounts?fields=id,name,account_status,currency,balance&access_token={token}`. Returns list of ad account dicts.

3. `async def list_campaigns(self, user_id: str, ad_account_id: str) -> list[dict]` -- GET `https://graph.facebook.com/v19.0/act_{ad_account_id}/campaigns?fields=id,name,status,objective,daily_budget,lifetime_budget,start_time,stop_time&access_token={token}`. Map status: ACTIVE->active, PAUSED->paused. Return list of campaign dicts.

4. `async def create_campaign(self, user_id: str, ad_account_id: str, name: str, objective: str = "OUTCOME_TRAFFIC", daily_budget: int = None, lifetime_budget: int = None, status: str = "PAUSED") -> dict` -- POST `https://graph.facebook.com/v19.0/act_{ad_account_id}/campaigns`. Body: name, objective, special_ad_categories=[], status=PAUSED (always paused), daily_budget (in cents) or lifetime_budget (in cents). Return campaign dict.

5. `async def update_campaign_status(self, user_id: str, campaign_id: str, status: str) -> dict` -- POST `https://graph.facebook.com/v19.0/{campaign_id}`. Body: status (ACTIVE/PAUSED). Map from internal names.

6. `async def update_campaign_budget(self, user_id: str, campaign_id: str, daily_budget: int = None, lifetime_budget: int = None) -> dict` -- POST `https://graph.facebook.com/v19.0/{campaign_id}`. Body: daily_budget (cents) and/or lifetime_budget (cents).

7. `async def get_campaign_insights(self, user_id: str, ad_account_id: str, start_date: str, end_date: str) -> list[dict]` -- GET `https://graph.facebook.com/v19.0/act_{ad_account_id}/insights?fields=campaign_id,campaign_name,impressions,clicks,spend,actions,action_values&time_range={'since':'{start_date}','until':'{end_date}'}&level=campaign&time_increment=1&access_token={token}`. Parse actions for conversions count. Return daily performance rows.

All HTTP calls use `httpx.AsyncClient` with 30s timeout. Meta budgets are in cents (multiply by 100 on write, divide by 100 on read).

**File 4: app/services/ad_budget_cap_service.py** -- New file.

Create `AdBudgetCapService(BaseService)`:

1. `async def get_cap(self, user_id: str, platform: str) -> float | None` -- Query ad_budget_caps for (user_id, platform). Return monthly_cap or None if not set.

2. `async def set_cap(self, user_id: str, platform: str, monthly_cap: float) -> dict` -- Upsert to ad_budget_caps with on_conflict="user_id,platform". Use AdminService for writes.

3. `async def check_budget_headroom(self, user_id: str, platform: str, proposed_daily_budget: float) -> dict` -- 
   - Get monthly_cap for platform
   - Sum daily_budget of all active ad_campaigns for this user+platform
   - Calculate committed = sum of active daily budgets × remaining days in month
   - Calculate headroom = monthly_cap - committed
   - proposed_monthly = proposed_daily_budget × remaining days
   - Return: {"allowed": bool, "monthly_cap": cap, "committed": committed, "headroom": headroom, "proposed_monthly": proposed_monthly, "message": str}
   - If not allowed: message = "This would exceed your ${cap}/mo {platform} cap (${committed} already committed). Increase your cap in Settings or reduce other campaigns."

4. `async def is_cap_set(self, user_id: str, platform: str) -> bool` -- Returns True if cap exists for user+platform.
