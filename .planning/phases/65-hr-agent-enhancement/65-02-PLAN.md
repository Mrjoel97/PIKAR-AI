    - Test: get_funnel_for_job returns conversion_rates between stages (e.g., applied->screening = 60%)
    - Test: get_funnel_summary returns all open jobs with their funnel counts
    - Test: funnel with zero candidates returns all stages at 0
    - Test: get_hiring_funnel tool returns success=True with funnel data
  </behavior>
  <action>
    1. Create `app/services/hiring_funnel_service.py`:
       ```python
       class HiringFunnelService(BaseService):
       ```
       - `async def get_funnel_for_job(self, job_id: str, user_id: str | None = None) -> dict`:
         - Queries `recruitment_candidates` grouped by status for the given job_id
         - Defines the funnel stages in order: ["applied", "screening", "interviewing", "offer", "hired"]
         - Also counts "rejected" separately (not in funnel but reported)
         - Returns: `{"job_id": ..., "stages": [{"name": "Applied", "count": N}, ...], "rejected": N, "total": N, "conversion_rates": {"applied_to_screening": 0.6, ...}}`
         - Conversion rate = next_stage_count / current_stage_count (0 if current is 0)
       - `async def get_funnel_summary(self, user_id: str | None = None) -> list[dict]`:
         - Queries all published/open jobs for the user
         - For each job, gets funnel counts
         - Returns list of `{"job_id": ..., "title": ..., "department": ..., "funnel": {...}}`

    2. Create `app/routers/recruitment.py`:
       - `router = APIRouter(prefix="/api/recruitment", tags=["Recruitment"])`
       - `GET /api/recruitment/funnel/{job_id}` -> calls `HiringFunnelService().get_funnel_for_job(job_id)`
       - `GET /api/recruitment/funnel` -> calls `HiringFunnelService().get_funnel_summary()`
       - Both endpoints require auth via `Depends(get_current_user_id)` and rate limiting via `@limiter.limit(get_user_persona_limit)`
       - Follow the same pattern as `app/routers/org.py`

    3. Add `get_hiring_funnel` tool to `app/agents/hr/tools.py`:
       ```python
       async def get_hiring_funnel(job_id: str = None) -> dict:
           """Get the hiring funnel visualization data for a specific job or all open positions.
           
           Args:
               job_id: Optional job ID. If omitted, returns funnel for all open positions.
           """
       ```
       - If job_id provided: calls `HiringFunnelService().get_funnel_for_job(job_id)`
       - If job_id is None: calls `HiringFunnelService().get_funnel_summary()`
       - Returns `{"success": True, "funnel": data}` or `{"success": False, "error": str(e)}`

    4. Write tests mocking Supabase responses. Test stage counting, conversion rates, empty funnel, and summary across multiple jobs.
  </action>
