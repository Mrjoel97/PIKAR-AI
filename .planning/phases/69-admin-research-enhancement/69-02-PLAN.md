    Create `app/services/billing_alert_service.py`:
    - Class `BillingAlertService(AdminService)`:
      - Constants: `COST_INCREASE_WARNING_THRESHOLD = 0.20` (20%), `COST_INCREASE_CRITICAL_THRESHOLD = 0.50` (50%)
      - `async def compute_cost_projection(self) -> dict[str, Any]`:
        1. Instantiate `ObservabilityMetricsService()` and call `project_monthly_ai_spend()` for current month projection.
        2. Compute prior month total: use `compute_ai_cost_by_day(first_day_of_prior_month, last_day_of_prior_month)` and sum all daily costs.
        3. Compute month-over-month change: `(projected - prior) / prior * 100` (handle prior=0 as "new usage").
        4. Get top cost drivers: call `compute_ai_cost_by_agent(mtd_start, now)` and sort by cost descending, take top 5.
        5. Determine severity: >50% increase = "critical", >20% = "warning", else None.
        6. Generate `plain_english_summary`:
           - "At current usage, this month's AI costs are projected to be ${projected:.2f} — {change_pct:.0f}% {higher/lower} than last month (${prior:.2f})."
           - If increase: "The main cost driver is {top_agent} at ${agent_cost:.2f} ({pct_of_total:.0f}% of total spend)."
           - If decrease or flat: "Costs are trending {flat/down} compared to last month."
        7. Return `{"mtd_actual": ..., "projected_full_month": ..., "prior_month_total": ..., "month_over_month_change_pct": ..., "top_cost_drivers": [...], "alert_recommended": bool, "severity": str|None, "plain_english_summary": str}`

      - `async def check_and_alert(self, admin_user_ids: list[str] | None = None) -> dict[str, Any]`:
        1. Call `compute_cost_projection()`.
        2. If `alert_recommended`, dispatch notification to each admin user (or a default admin list from `user_executive_agents` where persona='admin').
        3. Event type: `"billing.cost_projection_alert"`, payload includes title, body (plain_english_summary), severity.
        4. Return `{"alerted": bool, "projection": projection_result, "notifications_sent": int}`
