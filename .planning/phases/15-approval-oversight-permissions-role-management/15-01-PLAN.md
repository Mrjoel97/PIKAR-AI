       ```python
       def require_admin_role(min_role: str):
           async def _check(admin: dict = Depends(require_admin)) -> dict:
               admin_level = ROLE_HIERARCHY.get(admin.get("admin_role", ""), 0)
               required_level = ROLE_HIERARCHY.get(min_role, 99)
               if admin_level < required_level:
                   raise HTTPException(403, f"Requires {min_role} or higher")
               return admin
           return _check
       ```

    **Create app/routers/admin/approvals.py:**

    1. Create router = APIRouter(tags=["admin-approvals"])

    2. GET /approvals/all endpoint:
       - Depends: require_admin
       - Query params: status (optional, default "PENDING"), action_type (optional), user_id (optional UUID), limit (int, default 50), offset (int, default 0)
       - Query approval_requests table with service client, order by created_at DESC
       - Apply filters: .eq("status", status) if status, .eq("action_type", action_type) if action_type, .eq("user_id", user_id) if user_id
       - Use .range(offset, offset + limit - 1) for pagination
       - Return list of approval dicts with id, action_type, status, payload, created_at, expires_at, user_id

    3. POST /approvals/{approval_id}/override endpoint:
       - Depends: require_admin_role("senior_admin") -- junior_admin cannot override
       - Body: OverrideDecision(decision: str, reason: str | None)
       - Validate decision in ("APPROVED", "REJECTED")
       - Fetch current approval by id, verify status == "PENDING" (409 if not)
       - Check expires_at (mark EXPIRED if past due)
       - Atomic update: UPDATE approval_requests SET status=decision, responded_at=now, responder_ip=client_ip WHERE id=approval_id AND status='PENDING'
       - log_admin_action with source="admin_override", action="override_approval", target_type="approval", target_id=approval_id, details={decision, reason, original_action_type}

    4. GET /roles endpoint:
       - Depends: require_admin_role("admin") -- need admin+ to view roles list
       - Query user_roles table: SELECT user_id, role, created_at, updated_at WHERE role != 'user'
       - Return list of admin role dicts

    5. POST /roles endpoint:
       - Depends: require_admin_role("super_admin") -- only super can create admins
       - Body: CreateAdminRole(user_id: str, role: str)
       - Validate role in ('junior_admin', 'senior_admin', 'admin', 'super_admin')
       - Upsert into user_roles: INSERT ... ON CONFLICT (user_id) DO UPDATE SET role=..., updated_at=now()
       - log_admin_action with source="manual", action="create_admin_role"

    6. DELETE /roles/{target_user_id} endpoint:
       - Depends: require_admin_role("super_admin")
       - Delete from user_roles WHERE user_id=target_user_id AND role != 'user'
       - log_admin_action with source="manual", action="delete_admin_role"

    7. GET /roles/permissions endpoint:
       - Depends: require_admin_role("admin")
       - Query admin_role_permissions table: SELECT role, section, allowed_actions
       - Return grouped by role

    8. PUT /roles/permissions endpoint:
       - Depends: require_admin_role("super_admin")
       - Body: UpdateRolePermission(role: str, section: str, allowed_actions: list[str])
       - Validate role, section, and actions
       - Upsert into admin_role_permissions
       - log_admin_action with source="manual", action="update_role_permissions"
