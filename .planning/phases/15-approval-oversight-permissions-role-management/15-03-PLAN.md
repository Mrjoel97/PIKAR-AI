From Plan 01 - API endpoints (app/routers/admin/approvals.py):
```
GET /admin/approvals/all?status=PENDING&action_type=...&user_id=...&limit=50&offset=0
  -> [{id, action_type, status, payload, created_at, expires_at, user_id}]

POST /admin/approvals/{approval_id}/override
  Body: {decision: "APPROVED"|"REJECTED", reason: string|null}
  -> {success, status, message}

GET /admin/roles
  -> [{user_id, role, created_at, updated_at}]

POST /admin/roles
  Body: {user_id: string, role: "junior_admin"|"senior_admin"|"admin"|"super_admin"}
  -> {success, user_id, role}

DELETE /admin/roles/{user_id}
  -> {success}

GET /admin/roles/permissions
  -> [{role, section, allowed_actions}]

PUT /admin/roles/permissions
  Body: {role: string, section: string, allowed_actions: string[]}
  -> {success}
```
