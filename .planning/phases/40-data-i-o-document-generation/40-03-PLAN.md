       - Docstring explains: exports data from the specified table to a CSV file and returns a download URL. Supported tables: same as import.
       - Extract user_id from _get_request_scope()

       - Call DataExportService.export_table()
       - Return dict: {"status": "success", "url": signed_url, "filename": filename, "size_bytes": N, "widget": {type: "document", data: {documentUrl, title, fileType: "csv", sizeBytes}}}

       - Return the widget dict from DocumentService
