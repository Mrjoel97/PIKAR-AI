Key observation: The current code uses `asyncio.create_task` (fire-and-forget).
To capture the interaction_id, we must `await` instead.
But we also need to yield the final SSE event BEFORE the generator returns.
The `finally` block of the inner `try` (lines 1887-1941) runs while
the generator is still active and can still yield.
