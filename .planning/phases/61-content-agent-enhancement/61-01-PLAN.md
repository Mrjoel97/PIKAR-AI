<objective>
Add a one-shot fast path to the Content Agent so simple content requests (social posts, blog intros, emails) produce a ready-to-use draft in a single conversational turn without triggering the 10-stage creative pipeline.

Purpose: Currently every content request routes through creative brief -> concept exploration -> delegation to sub-agents, which is overkill for "write me a tweet about X." This plan adds a `simple_create_content` tool and instruction-level routing so the Content Director handles simple requests directly.

Output: New `simple_create_content` tool, updated agent instructions, unit tests.
</objective>
