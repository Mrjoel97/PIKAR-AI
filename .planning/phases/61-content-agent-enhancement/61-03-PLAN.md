<objective>
Build an automatic brand voice learning service that analyzes a user's content history (5+ pieces) to extract tone, vocabulary, and style patterns, then persists these learned patterns to the brand profile for application in future content.

Purpose: Users shouldn't have to manually describe their brand voice. After creating several pieces of content, the system should auto-detect their writing patterns (average sentence length, tone markers, common phrases, formality level) and apply them to all future content generation.

Output: New BrandVoiceService, learn_brand_voice tool, updated agent instructions, unit tests.
</objective>

    ```python
    class BrandVoiceService:
        """Analyzes user content history to extract and learn brand voice patterns."""

        MIN_CONTENT_PIECES = 5

        async def analyze_and_learn(self, user_id: str) -> dict:
            """Full pipeline: fetch history -> analyze -> persist to brand profile."""

        async def get_content_history(self, user_id: str, limit: int = 50) -> list[str]:
            """Fetch user's content texts from ContentService."""

        def analyze_content_samples(self, texts: list[str]) -> dict:
            """Pure function: analyze text samples, return voice features."""

        def extract_tone_markers(self, texts: list[str]) -> dict:
            """Extract: exclamation_rate, question_rate, emoji_rate, formality_score."""

        def extract_vocabulary_patterns(self, texts: list[str]) -> dict:
            """Extract: top distinctive words, average word length, jargon patterns."""

        def extract_sentence_patterns(self, texts: list[str]) -> dict:
            """Extract: avg_sentence_length, sentence_length_variance, short_sentence_ratio."""

        def build_voice_profile(self, tone: dict, vocab: dict, sentences: dict) -> dict:
            """Compile features into a voice profile with voice_tone, voice_personality, voice_examples."""

        async def persist_voice_to_brand_profile(self, user_id: str, voice_profile: dict) -> dict:
            """Save learned voice to brand profile via update_brand_profile."""
    ```

    Implementation details for `analyze_content_samples`:
    - Split each text into sentences using regex `r'[.!?]+\s+'`
    - Calculate `avg_sentence_length` = mean words per sentence across all samples
    - Calculate `formality_score` between 0 (very casual) and 1 (very formal) based on:
      - Presence of contractions (informal: lower score)
      - Emoji/emoticon usage (informal: lower score)
      - Average word length (longer words = more formal)
      - Exclamation mark frequency (more = less formal)
    - Extract `common_phrases`: 2-gram and 3-gram phrases that appear in 3+ content pieces
    - Build `tone_descriptors` list from formality + markers (e.g., ["conversational", "enthusiastic"] or ["professional", "authoritative"])

    Implementation for `persist_voice_to_brand_profile`:
    - Import and call `update_brand_profile` from `app.agents.tools.brand_profile`
    - Map voice_profile fields to brand_profile fields:
      - `voice_profile["tone_summary"]` -> `voice_tone` param
      - `voice_profile["personality_traits"]` -> `voice_personality` param
      - `voice_profile["example_sentences"]` (pick 3 representative sentences from content) -> `voice_examples` param
    - Pass `user_id=user_id` to scope to the correct user

    For tests: Use pure text samples (no DB needed for analysis tests). Mock `ContentService.list_content` and `update_brand_profile` for integration tests. Include realistic text samples that produce measurable patterns (e.g., 5 casual blog posts with contractions and exclamation marks should yield formality_score < 0.4).

    Use standard Python only (re, collections.Counter) -- no NLTK or spaCy dependency.

    ```python
    async def learn_brand_voice() -> dict:
        """Analyze the user's content history to learn their brand voice patterns.

        Requires at least 5 content pieces. Extracts tone, vocabulary, and style
        patterns and saves them to the brand profile for use in future content.

        Returns:
            Dictionary with learned voice profile or insufficient-data message.
        """

        from app.services.request_context import get_current_user_id

        try:
            user_id = get_current_user_id()
            service = BrandVoiceService()
            result = await service.analyze_and_learn(user_id)
            return result
        except Exception as e:
            return {"success": False, "error": str(e)}
    ```
