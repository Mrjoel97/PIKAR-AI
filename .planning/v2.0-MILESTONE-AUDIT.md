## Cross-Phase Integration

**18 cross-service connections verified.** 2 integration issues found:

### P1: GsdProgressBar always shows wrong stage (BLDR-04)
- **Location:** `frontend/src/app/app-builder/[projectId]/layout.tsx` line 8
- **Issue:** Server component fetches `GET /projects/{id}` without auth header → 403 → falls back to `'research'`
- **Fix:** Convert to client-side hook, or forward session cookie in server fetch

### P2: Prompt enhancement bypassed in HTTP flow (FOUN-03)
- **Location:** `app/routers/app_builder.py` line 253 (`_build_generation_prompt`)
- **Issue:** The router's helper builds prompts without calling `enhance_prompt()`. The prompt enhancer only runs via orphaned ADK tool wrappers.
- **Fix:** Call `await enhance_prompt(prompt)` in the generate-screen router handler

### P3: APP_BUILDER_TOOLS orphaned (tech debt)
- **Location:** `app/agents/tools/app_builder.py` line 172
- **Issue:** ADK tool wrappers defined but never registered with any agent
- **Fix:** Register with ExecutiveAgent or document as superseded by HTTP pipeline
