# Nano Banana - Visual Asset Generation

This skill specializes in generating prompts and strategies for creating high-impact, artistic visual assets for web interfaces.

## Core Philosophy

- **Vibrancy**: Assets should pop. High saturation, deep contrast, and vivid colors.
- **Surrealism**: Avoid stock photo realism. Embrace abstract shapes, floating elements, and dreamlike compositions that metaphorically represent technical concepts.
- **Cohesion**: Ensure all generated assets share a consistent lighting engine (e.g., "volumetric purple lighting from top-left") and texture palette.

## Prompting Strategy

When generating images for the landing page:

1.  **Subject**: Define the core concept (e.g., "Artificial Intelligence", "Orchestration").
2.  **Style**: "3D render, octane render, unreal engine 5, glassmorphism, translucent materials, iridescent textures".
3.  **Lighting**: "Cinematic lighting, rim lights, neon accents, cyber-noir atmosphere".
4.  **Composition**: "Minimalist background, centered subject, wide angle, 8k resolution".

## Usage in Frontend

- Use these images as "Hero" backgrounds (with overlay opacity).
- Use as floating "glass" elements in feature sections.
- Use as distinct card headers.

## Example Prompts

**Hero Background:**
> "Abstract representation of a digital neural network, glowing nodes connected by beams of light, deep blue and purple gradient background, 3D glass texture, depth of field, 8k, highly detailed, futuristic."

**Feature Icon (Banana Style):**
> "A stylized, translucent 3D banana glowing with neon energy, floating in a void, cybernetic details, yellow and cyan lighting, minimalist."
