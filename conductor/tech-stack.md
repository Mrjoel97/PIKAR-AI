# Tech Stack

## Backend

*   **Language:** Python
*   **Framework:** FastAPI
*   **Agent Framework:** Google Agent Development Kit (ADK)

## Frontend

*   **Framework:** Next.js
*   **Language:** TypeScript
*   **UI Library:** React
*   **Styling:** Tailwind CSS

## Database

*   **Database:** PostgreSQL (via Supabase)

## Infrastructure

*   **CI/CD:** Google Cloud Build
*   **Infrastructure as Code:** Terraform
*   **Caching:** Redis

## Architecture

*   **Pattern:** Monorepo
*   **Agent Architecture:** Hierarchical agent system with a central "Executive Agent"
*   **Communication:** Agent-to-Agent (A2A) protocol
